## 💡 The Science
This engine simulates **Newtonian Gravity**. Every object attracts every other object using the formula:

$$ F = G \frac{m_1 m_2}{r^2} $$

## 🎮 How to Play
1.  **Solar System:** Loads a preset Star and Planets.
2.  **Slingshot:** Click anywhere and drag your mouse to launch a new planet.
3.  **Time Control:** Use the sliders to speed up time or increase gravity.

## 🛠️ Tech Stack
